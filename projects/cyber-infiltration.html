PROJECT 03 · AI NATIVE GAME

光球潜入：移动端体感游戏

结合 H5 Canvas、手机体感控制、GLM API 和 Cloudflare Serverless 的 Web 游戏，用 AI 生成剧情反馈和结算文案。

H5 CanvasJavaScriptGLM APICloudflare WorkersD1

项目目标

做一个在移动端可玩的体感 Web 游戏：玩家通过设备动作参与交互，系统根据表现生成个性化反馈，并支持排行榜和数据持久化。

我的工作

  • 使用 Claude Code 辅助完成代码实现、调试和 AI Native 开发流程。
  • 设计 Prompt 映射逻辑，把玩家行为转化为可用于 GLM API 的输入。
  • 接入 GLM API 生成结算文案，并配置 fallback 机制保证主流程稳定。
  • 使用 Cloudflare Pages、Workers 和 D1 构建轻量后端、排行榜和数据持久化。

项目亮点

它不是简单地把 AI 放到页面里，而是把 AI 放进玩法反馈：用户完成一次交互后，模型输出成为体验的一部分，让每次结算更像“个人化回应”。
