PROJECT 01 · ROBOTICS SYSTEM

XF-ROBOT-UCAR01 智能小车

基于 ROS1、Python Flask、WebSocket、Cartographer、RViz 与 Linux 的智能车 Web 控制系统，将实体机器人、Web 驾驶舱、地图导航和运行日志联动起来。

ROS1Python FlaskWebSocketCartographerRVizLinux

项目目标

面向 XF-ROBOT-UCAR01 智能小车，构建可在浏览器中操作的控制台，完成远程驾驶、导航目标下发、状态监控、地图展示、巡逻和日志分析等功能。

我的工作

  • 使用 Claude Code 辅助完成 ROS/Linux 代码编写、服务部署、日志排查和联调问题定位。
  • 开发 Web 驾驶舱，支持底盘控制、STOP、导航目标下发、AMCL 位姿展示和摄像头预览。
  • 通过 SSH 连接实体车环境，完成服务启动、异常排查、运行验证和稳定性调整。
  • 梳理系统联调流程，将硬件、ROS 节点、WebSocket 通信和前端交互串联成可复现流程。

技术难点

项目难点不只在页面实现，而在真实硬件场景下的不确定性：ROS 节点状态、地图构建、网络通信、服务部署和传感器数据都可能影响控制台表现。因此项目重点是把日志、运行状态和操作反馈做成可排查的工程链路。
